---
title: Config
---

Configure RSA CLI.

Get config `[key]`:

```sh
$ rsa config get [key]
```

Set config `[key]` to `[value]`:

```sh
$ rsa config set [key] [value]
```

Delete config `[key]`:

```sh
$ rsa config set [key]
# or:
$ rsa config set [key] unset
```

List all config:

```sh
$ rsa config list
```
